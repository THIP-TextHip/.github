# 서비스 소개

THIP(이하 ‘띱’)은 독서를 완독 중심의 결과물이 아닌 **읽는 과정 자체를 기록하고 공유하며 함께 이어가는 경험**으로 확장하는 커뮤니티형 독서 기록 플랫폼입니다.

사용자는 책을 완독하지 않아도 페이지 단위로 감상을 남길 수 있으며 피드와 모임방을 통해 다른 사용자들과 소통할 수 있습니다.

또한 취향 기반 추천(피드, 모임방, 책)을 통해 독서를 혼자 하는 활동이 아닌 **관계와 참여가 있는 문화 경험**으로 만들어갑니다.

이러한 흐름 속에서 띱은 ‘텍스트 힙(Text Hip)’ 문화를 기반으로 독서를 ‘결과 중심의 활동’이 아닌 **과정 중심의 지속 가능한 문화 경험**으로 확장하고자 합니다.

# 개발 배경

최근 독서는 SNS를 중심으로 ‘힙한 취향 활동’으로 인식되며 독서 콘텐츠에 대한 관심은 증가하고 있지만,

실제 독서 지속률과 완독률은 여전히 낮은 상태에 머물러 있습니다.

저희는 이러한 문제의 원인을 파악하기 위해 20~30대 성인 109명을 대상으로 유저 리서치를 진행했고

그 결과 다음과 같은 한계를 도출했습니다.

**❌ 기존 독서 플랫폼의 한계**

- 완독 이후 리뷰 중심의 기록 구조로 인한 참여 장벽
- 정형화된 서평 작성 방식으로 발생하는 심리적 부담
- 시간·장소·참여 부담이 큰 오프라인 독서 모임 구조

⬆️ 해결방안??

- 완독 여부와 관계없이 페이지 단위로 가볍게 기록할 수 있는 구조
- 글·이미지, 투표 등 다양한 형식으로 부담 없이 독서 경험 공유
- 혼자 하는 독서가 아닌 함께하는 독서 경험으로 제공!
